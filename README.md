# GL-neuronized-svg

Geometry-Aware Bayesian Detection of Spatially Variable Gene in Three Dimensions

## Project goals

-   Build graph Laplacian basis from 3D spatial coordinates
-   Fit graph-spectral Bayesian regression model
-   Implement MCMC updates for $\mu_g, \omega_g, \xi_g, \tau_g^2, \sigma_g^2$
-   Use FDR to detect svg
